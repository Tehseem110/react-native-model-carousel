# react-native-model-carousel

A React Native library for rendering interactive 3D GLB models in a smooth, customizable carousel powered by Three.js

## Installation


```sh
npm install react-native-model-carousel
```


## Usage


```js
import { multiply } from 'react-native-model-carousel';

// ...

const result = multiply(3, 7);
```
